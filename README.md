# GRC System - Governance, Risk & Compliance Platform

Comprehensive governance and compliance management system for Kuwait government entities.

## 🏗️ Project Structure

```
GRCSystem/
├── Backend/
│   ├── GRC.API/
│   ├── GRC.Application/
│   ├── GRC.Domain/
│   ├── GRC.Infrastructure/
│   └── GRC.sln
├── Frontend/
│   └── grc-web/
├── Mobile/
│   └── grc-mobile/
└── Database/
    └── 01-CreateDatabase.sql
```

## 🚀 Tech Stack

- **Backend**: ASP.NET Core 8, Entity Framework Core, SQL Server 2022
- **Frontend**: React 19, Material-UI, Redux Toolkit
- **Mobile**: React Native, Expo
- **Auth**: JWT Authentication

## ✨ Features

✅ Compliance Matrix Management
✅ CMA Violations Tracking
✅ Ministers Database with Timeline
✅ National Assembly Members
✅ Citizenship Cases Management
✅ Dashboard & KPIs
✅ Audit Logs
✅ Role-based Access Control
